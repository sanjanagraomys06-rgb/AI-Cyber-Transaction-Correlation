# Demo

## Project Name
IntelliGuard AI

## Theme
AI-Driven Correlation of Cybersecurity Telemetry & Transactional Behaviour

## Objective
This project uses Artificial Intelligence to correlate cybersecurity telemetry with banking transaction behaviour to detect fraudulent activities in real time.

## Demo Workflow

1. User logs into the system.
2. Cybersecurity telemetry (firewall logs, SIEM alerts, login events) is collected.
3. Banking transaction data is received.
4. The AI Correlation Engine analyzes both datasets.
5. A risk score is generated.
6. Suspicious transactions are flagged.
7. Alerts are displayed on the dashboard.
8. The security analyst can approve or block the transaction.

## Expected Output

- Real-time fraud detection
- AI-generated risk score
- Live security alerts
- Explainable AI insights
- Fraud analytics dashboard
